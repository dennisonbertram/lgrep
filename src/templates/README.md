# mgrep Templates

This directory contains templates used by the `mgrep install` command to integrate mgrep with Claude Code.

## Files

- **skill.md** - Claude Code skill that teaches Claude how and when to use mgrep
- **mgrep-check.sh** - SessionStart hook script that auto-starts watchers
- **claude-md-section.md** - Section to add to project CLAUDE.md files

## Usage

These templates are automatically loaded and used by the `mgrep install` command. They are embedded in the built distribution and do not need to be manually copied.
